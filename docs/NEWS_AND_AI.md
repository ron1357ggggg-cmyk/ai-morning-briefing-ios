# 新聞與 AI

## 時間窗

產品需求為前一日 09:00 至當日 09:00。時間窗應由後端以
`Asia/Taipei` 計算，避免裝置時區造成不一致。由於通知時間是 08:30，
08:30 產生的版本只能整理到當下；09:00 後重新整理才使用完整時間窗。

## 配額

- 總數 10 則
- 台灣至少 3 則
- 國際至少 7 則
- 排除娛樂八卦

## 每則格式

- 標題
- 三行內摘要
- 為何重要
- 分類
- 來源與原文網址

## 安全與品質

- AI 摘要必須可追溯到原始來源。
- 不得由單一模型憑記憶產生新聞。
- 政治、戰爭與公共衛生消息需交叉比對。
- API key 由後端保存，不進入 App。

## MVP 即時來源

- 使用 Google 新聞 RSS，不需要 API key。
- 台灣 query 取 3 則，國際／AI／科技／財經／政治 query 取 7 則。
- RSS 原文網址可由新聞詳情頁開啟。
- RSS 摘要資訊有限，目前使用本機模板，不宣稱為 AI 產生摘要。
- 未來接入後端後，才執行來源授權、交叉比對、去重與 OpenAI 繁中摘要。
